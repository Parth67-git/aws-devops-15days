\# AWS DevOps 15-Day Journey (Free Tier)



\## Goal

Become job-ready in AWS DevOps by building a real CI/CD + IaC portfolio project.



\## Progress Tracker

\- Day 1 ✅ AWS account setup + billing alerts

\- Day 2 ✅ EC2 + SSH + Nginx

\- Day 3 ✅ Versioned deployment structure + Nginx root fix

\- Day 4 ⏳ Git + GitHub + repo structure



\## Repo Structure

\- app/      -> Application code (versioned releases)

\- scripts/  -> Deployment scripts (deploy.sh etc.)

\- docs/     -> Notes PDFs + screenshots

\- app/releases/release-001

\- app/releases/release-002
